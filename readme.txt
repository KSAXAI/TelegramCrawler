# Telegram Crawler Bot

### توضیحات پروژه
این پروژه یک ربات کرالر برای تلگرام است که به منظور **استخراج داده‌ها** از گروه‌ها یا کانال‌های تلگرام طراحی شده است. این ابزار می‌تواند برای جمع‌آوری اطلاعات پیام‌ها، کاربران، و سایر داده‌های مرتبط مورد استفاده قرار گیرد.

### ویژگی‌ها
- استخراج پیام‌ها و اطلاعات کاربران
- پشتیبانی از گروه‌ها و کانال‌ها
- خروجی پیام ها با فرمت فایل Sqlite و رسانه ها در پوشه خودش با فرمت دریافتی

### پیش‌نیازها
قبل از اجرای این پروژه، موارد زیر باید روی سیستم شما نصب شده باشند:
- **Python 3.7 یا بالاتر**
- کتابخانه‌های زیر:
  - `telegram`
  - `Sqlite3`
